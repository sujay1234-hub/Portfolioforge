05 — FAQ

Common Questions

Everything you need to know before placing your order.

Do I actually get the source code?
+
Yes, every plan includes full source code. You own it completely — modify it, learn from it, host it anywhere, no strings attached.
What if I need changes after delivery?
+
Animated and Premium plans include 1–2 free revision rounds. Basic plan can be revised for a small fee. All revision requests are handled within 12 hours.
How exactly does the 24-hour guarantee work?
+
The 24-hour clock starts from payment confirmation. If your portfolio isn't delivered within 24 hours for any reason, you get a full refund — no questions asked.
What information do I need to provide?
+
Your name, skills, 2–4 project descriptions (with tech stack), GitHub link (optional), and any design preferences. The more detail you give, the better the result.
Which payment methods are accepted?
+
UPI (GPay, PhonePe, Paytm, BHIM) and bank transfer. A secure payment link is sent after order submission.
Can you deploy the site for me?
+
Yes! Add the "Custom Domain Setup" add-on during checkout and we'll handle full Vercel/Netlify deployment with your domain name.
Is this suitable for non-developers too?
+
Absolutely. We build portfolios for designers, students, freelancers, and startup founders — not just developers. Choose the style that fits your field.
What tech stack is used?
+
Basic: Pure HTML/CSS/JS. Animated: HTML + CSS animations + vanilla JS. Premium: React/Next.js with GSAP/Three.js. You can request a specific stack too.
